# CheeseAni Skill Certification

[**在线使用**](https://cheeseani.github.io/CheeseAni-Skill-Certification/)

## 项目介绍

### 功能

本项目用于存储本组考核通过的记录

项目存储的是 HASH，所以不用担心信息泄露

> HASH 从以下三个字符串相加后生成
>
> 1. ID 用于向所有方证明自己的身份，可以设为自己用于公开的社交账号
> 2. Staff 名用于证明自己在字幕制作的中的身份
> 3. 如果不希望任何人通过遍历字典反推出自己的信息，可以设置密钥

### 目的

本项目想让所有人都有一个难度和内容相对统一的字幕制作技能认证途径

### 认证

考核通过后，在本项目的在线使用页面中按照自己的需求填写，将生成的 HASH 都复制给考核员，并告知考核员该 HASH 绑定的 ID，考核员会将 HASH 添加到这个项目中  
（绑定的 ID 必须是自己账号的 ID，以防止替考等行为）

如果不希望任何人知道自己的任何账号，则可以将 ID 填为 staff 相同值  
（此时考核员仅允许添加单个认证记录）

如果不在意第三者知道自己获取了此认证，则无需添加密钥

如果希望明文保存，将密钥设为字符串 `null`，生成的值就会是明文

添加密钥后，计算 HASH 的时间可能会有十几秒或更久

同一个人可以添加多个认证记录，例如绑定不同的 ID，还可以设置不同的密钥，让不同群体根据自己提供的信息查证  
（考核员需确认所有 HASH 的 ID 都是同一个人的自用账号）

需要修改或删除自己的认证记录时，同样需要将 ID 和生成的 HASH 都复制给考核员

可以让考核员在认证中加上自定义备注，例如写上自己的博客主页地址，可以自定义任何前端代码

## 考核介绍

任何人都可以直接参加任何级别的考核，不局限于本组成员

可以通过本组公开的交流群联系考核者参加考核

注意: **所有考核标准为本组制定的标准，可能非通用标准**

***具体考核待完善，目前仅规划了大致方向***

### 时轴

1. #### 初级

   掌握基本的 Aegisub 打轴技术，通过时轴考核题

1. #### 中级

   能精准处理各种闪轴，通过更复杂的时轴考核题

1. #### 高级

   能快速高质量打轴，拥有相对较多的项目经验，由考核员根据实际情况判断

### 后期

1. #### 初级

   掌握 Aegisub 大部分技巧，了解 ASS 所有标签和大部分常见特性，通过后期考核题

2. #### 中级

   熟练使用 Aegisub，了解基本的 Lua 语法，会使用 Mocha 制作追踪字幕，能精准使用所有常见的标签效果，通过涉及复杂效果的后期考核题

   时轴水平至少初级

3. #### 高级

   基本掌握 Lua 语法，拥有相对较多的项目经验，由考核员根据实际情况判断

   时轴水平至少中级
