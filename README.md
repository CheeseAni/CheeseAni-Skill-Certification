# CheeseAni Skill Certification

[**在线使用**](https://cheeseani.github.io/CheeseAni-Skill-Certification/)

## 项目介绍

### 功能

本项目用于存储本组考核通过的记录

项目存储的是 HASH，所以不用担心信息泄露

> HASH 从以下三个字符串相加后生成
>
> 1. ID 用于向所有方证明自己的身份，可以设为自己用于公开的社交账号
> 2. Staff 名用于证明自己在 ASS 的 staff 中的身份
> 3. 如果不希望任何人通过遍历字典反推出自己的信息，可以设置密钥

### 目的

本项目想让所有人都有一个难度和内容相对统一的字幕制作技能认证途径

### 认证

考核通过后，在本项目的在线使用页面中按照自己的需求填写，将生成的 HASH 都复制给考核员，并告知考核员该 HASH 绑定的 身份 ID，考核员会将 HASH 添加到这个项目中  
（绑定的 身份 ID 必须是自己账号的 ID）

如果不在意第三者知道自己

同一个人可以添加多个 HASH，例如设置多个不同的密钥，让不同群体根据自己提供的密钥验证自己的认证  
（考核员需确认所有 HASH 的 身份 ID 都是同一个人的个人账号）

需要修改自己的认证记录时，同样需要将生成的 HASH 都复制给考核员

可以让考核员在认证中加上自定义备注，例如写上自己的博客主页地址

## 考核介绍

任何人都可以参加考核，不局限于本组成员

可以通过本组公开的交流群联系考核者参加考核

注意: **所有考核标准为本组制定的标准，可能非通用标准**

***具体考核待定，目前仅写了大致方向***

### 时轴

1. #### 初级

   掌握基本的 Aegisub 打轴技术，通过初级时轴考核题

1. #### 中级

   能精准处理各种闪轴，通过中级时轴考核题

1. #### 高级

   能快速高质量打轴，拥有相对较多的项目经验，由考核员根据实际情况判断

### 后期

1. #### 初级

   掌握 Aegisub 大部分技巧，了解 ASS 所有标签和大部分常见特性，通过初级后期考核题

2. #### 中级

   熟练使用 Aegisub，了解基本的 Lua 语法，会使用 Mocha 制作追踪字幕，能精准使用所有常见的标签效果，通过中级时轴考核题

3. #### 高级

   拥有相对较多的项目经验，由考核员根据实际情况判断
